About

Creating a AWS user group in Gibraltar was always going to be a challenge due to its very small size. The number of people engaged in using AWS here on the rock is small especially in comparison to some of the larger user group jurisdictions around the globe. In addition Gibraltar is very heavy with sectors such as finance and law, not so much in terms of technology. Rather than see this as an obstacle we saw it as an opportunity and thanks to a small core set of AWS enthusiasts here in Gibraltar we have an AWS user group up and running.

We could not look to rely on drawing out regular speakers from tech companies as a lot of the worldwide user groups do, the volume of tech companies using AWS and employees engaged with AWS is just not not here in Gibraltar. However we do hope to change that. In that sense we decided on a slightly different approach and one in which we would hope to encourage and provide a friendly forum for new users to AWS and engage those who are more experienced. We have embarked on a couple of open source projects that we hope to hack out over time with the aim of hands on experience using and creating AWS technologies.

Meetup

In the short time we have been up and running we have managed to get together once a month. Usually we meet on the first Wednesday of the month. The meetup is an informal affair where we usually enjoy a few drinks and discuss the topic of the night. We also discuss the open source project we are currently working on and attempt to progress that. Its a great opportunity for new people to AWS to come along and ask questions or for the more experienced to help others, technically challenge each other, swap notes and share ideas or just enjoy a beer together! You are always more than welcome to drop by and more information can be found on the meetup page.

AWS User Groups

AWS User Group Gibraltar is part of a large family of AWS dedicated user groups from all around the world. The large network of user groups gather on a regular basis to engage with all kinds of people engaged with AWS or looking to start out with AWS to answer questions, share ideas, and learn about new services and best practices.

More information about this growing dynamic community can be found on the AWS User Group homepage.

Projects

We are currently kicking off our first open source collaborative project 'hashtag Gibraltar'. We wanted to embark on a project with a focus on learning and providing entry level examples of key AWS services. This as well as creating something that would be relevant to the context of Gibraltar. The group came up with the idea of a Gibraltar based twitter monitor for Gibraltar.

The project got under way early 2017 and we hope to post more about that here in the coming months.

All our projects will be made available on GitHub, please feel free to contribute!

You can find us @ GitHub.
